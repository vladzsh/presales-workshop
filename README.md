# Discovery Workshop: Technical Excellence in Sales

Knowledge base з серії воркшопів, присвячених методології Discovery та архітектурному проєктуванню на етапі pre-sale.

Замітки структуровані як Obsidian vault з `[[wikilinks]]` між темами.


## Структура

### Процес Pre-sale та Discovery
- Цілі: продаж, допродаж, формування іміджу, підготовка рішення
- Ролі в Solution Group: Solution Architect, Business Analyst, Technical Specialist, Project Coordinator
- План Discovery (10 днів)

### Архітектурний підхід
- Якість вимог (NFR): Performance, Scalability, Availability, Security, Maintainability & Usability
- Інструменти аналізу: Utility Tree, ATAM, SCR Framework
- Документування: C4 Model (L1–L4)
